# AI & Data Science Job Market Analysis

## Project Overview
This project performs Exploratory Data Analysis (EDA) on a global AI & Data Science job dataset using Python.

The objective of this project is to analyze:
- Job role distribution
- Salary trends
- Remote work trends
- Country-wise hiring patterns
- Skill demand
- Experience level distribution
- Correlation between technical skills and salary

---

## Dataset Information
- Total Records: 10,345
- Total Columns: 19

### Features Used
- Job Title
- Company Industry
- Country
- Remote Type
- Experience Level
- Education Level
- Technical Skills
- Salary
- Hiring Urgency
- Job Openings

---

## Technologies Used
- Python
- Pandas
- Matplotlib
- Seaborn
- Jupyter Notebook

---

## Key Insights
- AI Engineer and Machine Learning Engineer roles showed the highest average salaries.
- Cloud, Machine Learning, and Deep Learning were among the most in-demand skills.
- Remote, Hybrid, and Onsite jobs were almost equally distributed.
- Germany, Singapore, and Canada showed high hiring activity.
- Deep Learning and Machine Learning skills showed positive correlation with salary.
